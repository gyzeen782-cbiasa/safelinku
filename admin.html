Cara Pasang Iklan Adsterra

Ikuti langkah berikut setelah akun Adsterra disetujui

1 Login Adsterra & pilih format iklan
Buka publishers.adsterra.com → Websites → pilih website → Add Format. Pilih Popunder atau Social Bar untuk hasil terbaik di link shortener.
2 Salin kode script iklan
Adsterra akan memberikan kode script. Salin kode tersebut.
3 Paste di file link.html & reveal.html
Buka file link.html dan reveal.html, cari komentar bertuliskan Paste kode iklan Adsterra di sini, lalu paste script dari Adsterra di sana.
4 Commit ke GitHub → Vercel auto-deploy
Setelah paste kode iklan, commit file tersebut ke GitHub. Vercel akan otomatis deploy perubahan. Selesai! 🎉
